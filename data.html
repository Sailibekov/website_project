About the Data

This dataset offers an overview of the key factors that influence student academic performance. It includes a wide range of variables such as study habits, class attendance, parental involvement, and other behavioral and socio-demographic indicators. By capturing these diverse elements, the dataset enables a deep analysis of student performance and helps uncover insights into the educational process.

Data Structure

Variable Type Description
gender Categorical Student's gender
attendance Continuous Percentage of classes attended
hours_studied Continuous Weekly study hours outside class
parental_involvement Categorical Level of parental involvement (Low, Medium, High)
sleep_hours Continuous Average number of hours of sleep per night
access_to_resources Categorical Availability of educational resources (Low, Medium, High)
exam_score Continuous Final exam score

Data Source

The dataset is a fabricated compilation based on real-world data available on Kaggle. It contains detailed information on 6,607 students, featuring 20 different variables. This comprehensive dataset is designed to help analyze the multifaceted aspects of academic performance.

The source data has been curated to include a diverse set of indicators ranging from attendance and study habits to socio-demographic factors, offering a robust foundation for insights into student outcomes.
